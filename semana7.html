Sábado 14 de Octubre

Vimos Arduino IDE para poder empezar con la programación del proyecto en esta plataforma. Es importante mencionar que se pueden escribir varias líneas de código y cada una pertenece a un tipo. Se encuentran las funciones que permiten hacer cálculos para controlar un componente como Arduino, la estructura compone a los mismos elementos que se encuentran en el código y las variables permiten almacenar un dato debido a que es un espacio de memoria. No olvidar de compilar el código y verificar la conexión del Arduino para que pueda cargar el código sin ningún problema.
